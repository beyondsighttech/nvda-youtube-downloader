# NVDA YouTube Downloader

A fully accessible NVDA add-on to download videos and audio from YouTube.

## Features
- **Download Video (MP4) or Audio (MP3)**: Choose your preferred format.
- **Playlist Support**: Download entire playlists or select specific videos.
- **Accessible UI**: Designed for screen reader users with easy navigation and keyboard shortcuts.
- **Download Management**: Stop, Retry, and Remove downloads easily from the list.
- **Smart Clipboard**: Automatically detects YouTube links on the clipboard.
- **Trimming**: Download specific sections of a video (Start/End time).

## Installation
1. Download the latest `.nvda-addon` release from the [Releases](https://github.com/beyondsighttech/nvda-youtube-downloader/releases) page.
2. Open the file to install it with NVDA.
3. Restart NVDA when prompted.

## Usage
- Press `NVDA+Shift+Y` to open the downloader dialog.
- Paste a YouTube link (or let it auto-detect).
- Choose Format and Quality.
- Click "Add to Download Queue".
- Manage downloads in the "Active Downloads" list using Up/Down arrows and the buttons below.

## Requirements
- NVDA 2019.3 or later.
- Internet connection.
