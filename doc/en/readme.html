NVDA YouTube Downloader

A fully accessible add-on for NVDA to download videos and audio from YouTube seamlessly.

Features

  • Download Video or Audio: Choose between MP4 (Video) and MP3 (Audio) formats.
  • Playlist Support: Download entire playlists or select specific videos from a playlist.
  • Quality Selection: Select your preferred video resolution (1080p, 720p, etc.) or audio bitrate.
  • Trimming: Download only a specific part of a video by setting Start and End times.
  • Accessible UI: Fully compatible with NVDA, featuring a list of active downloads with keyboard navigation.
  • Smart Clipboard: Automatically detects YouTube links when you open the downloader.
  • Download Management: Stop, Retry, or Remove downloads easily.

Installation

  1. Download the latest .nvda-addon file.
  2. Open the file (press Enter on it).
  3. NVDA will ask if you want to install the add-on. Select Yes.
  4. Restart NVDA when prompted.

Usage

  1. Open Downloader: Press NVDA+Shift+Y.
  2. Enter URL: Paste a YouTube video or playlist link. If you have a link on your clipboard, it will be pasted automatically.
  3. Select Format: Choose MP3 (Audio) or MP4 (Video).
  4. Select Quality: Choose your desired quality (e.g., Best, 1080p, 320kbps).
  5. Trimming (Optional): Enter Start and End times (HH:MM:SS) if you only want a clip.
  6. Start Download: Tab to the "Add to Download Queue" button and press Enter.

Managing Downloads

The "Active Downloads" list shows all your current downloads.

  • Use Up/Down Arrow keys to navigate the list.
  • Stop: Stops the selected active download.
  • Retry: Retries a failed or stopped download.
  • Remove: Removes a completed or stopped download from the list.
